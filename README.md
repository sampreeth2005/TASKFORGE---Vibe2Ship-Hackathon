# 📖 Overview

TaskForge is an AI-powered productivity companion designed to transform how users manage tasks, deadlines, and commitments.

Traditional productivity tools depend on passive reminders that users often ignore. TaskForge takes a proactive approach by leveraging Google's Gemini AI to understand priorities, generate actionable plans, optimize schedules, and provide personalized productivity coaching.

Whether you're a student preparing assignments, a professional managing meetings, or an entrepreneur balancing multiple responsibilities, TaskForge helps ensure that important work gets completed before deadlines become emergencies.

---

# ❗ The Problem

Every day, people miss:

- Assignment submissions
- Project deadlines
- Meetings
- Interviews
- Bill payments
- Personal goals
- Habit streaks

Most reminder apps only notify users.

They do not help users:

- Decide what to do first
- Break down complex tasks
- Plan execution
- Stay accountable

TaskForge bridges this gap by acting as a personal AI productivity coach.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/taskforge.git
```

## Navigate

```bash
cd taskforge
```

## Install Dependencies

```bash
npm install
```

## Configure Environment

Create:

```env
GEMINI_API_KEY=your_api_key
```

## Start Development Server

```bash
npm run dev
```
